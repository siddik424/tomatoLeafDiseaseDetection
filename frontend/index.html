How it works
Research-Grade Deep Learning
for Plant Health Monitoring
This system implements a two-phase VGG16 transfer learning pipeline trained on the PlantVillage tomato dataset. Phase 1 trains only the classification head while the VGG16 backbone stays frozen. Phase 2 fine-tunes the last convolutional block (block5) with a 10× lower learning rate for optimal generalisation.
🏗️
VGG16 Architecture
ImageNet-pretrained backbone with custom Dense(512) → Dense(256) classification head, BatchNorm, and Dropout regularisation.
⚖️
Class-Weight Balancing
Sklearn's compute_class_weight (balanced) counters dataset skew across all 10 classes during training.
🔬
Grad-CAM Explainability
Gradient-weighted Class Activation Maps highlight which leaf regions drove the model's prediction.
📊
Rigorous Evaluation
Held-out test set with Accuracy, F1, AUC-ROC, Cohen's Kappa, MCC, and per-class analysis for journal reporting.
🔄
Heavy Augmentation
Rotation, shift, shear, zoom, flips, and brightness jitter improve generalisation to real-world field conditions.
🛡️
Not-Tomato Detection
Confidence threshold + green-channel heuristic flags non-tomato images before a false diagnosis is shown.
